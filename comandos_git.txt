git init . 	-> Cria um grupo git na pasta atual
git status 	-> Mostra o status do trabalho
git add nome_arquivo 	-> Adiciona novo arquivo para ser commitado
git commit -m "msg" 	-> Faz commit do que estava para ser commitado
git log		-> Mostra todo o histórico de commit


##Fluxo de trabalho##

1. criar o diretório 
2. iniciar o git dentro do diretório (git init .)
3. edita ou cria arquivos
4. git status
5. git add <nome do arquivo>
6. git status
7. git commit -m "Mensagem"
8. git status
9. volta para o passo 3
